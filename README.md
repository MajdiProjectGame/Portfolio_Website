# 🌐 Portfolio Website

A personal portfolio website showcasing my projects, skills, and experience.

## 🚀 Features

- Responsive design
- About Me section
- Project showcase
- Smooth animations
- Background music support
- Custom styling with CSS

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- SVG Graphics
